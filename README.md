# web3.js - Ethereum JavaScript API

This is the Ethereum [JavaScript API][docs]

You need to run a local or remote TOP node to use this library.

Please read the [documentation][docs] for more.

## Installation

### Node

```bash
npm install topjs
```

### Node

```bash
npm install topjs
```

### browser

use script

```bash
npm build browser
```
